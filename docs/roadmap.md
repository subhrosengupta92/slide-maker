# Product Roadmap

## Phase 1: MVP (now)
- Prompt-to-PPTX generation through CLI.
- Design-forward templates and theme system.
- Optional OpenAI planner with deterministic fallback.

## Phase 2: Quality engine
- Auto layout scoring (readability, balance, hierarchy).
- Iterative agent loop: draft -> critique -> revise.
- Brand profile support (logo, color, typography, voice).

## Phase 3: Data + charts
- Structured data extraction from docs/CSV.
- Native editable chart support in PPTX.
- Table and grid modules for board updates.

## Phase 4: Productization
- API service and web app wrappers.
- Team workspaces, reusable slide blocks, and approval workflow.
- Export to Google Slides and keynote-compatible variants.
